# Core C++ Systems: Automation & Security

A collection of production-ready, console-based C++ applications demonstrating robust core programming, low-level system streams, and Object-Oriented design principles.

## 📌 Projects Included

### 1. Secure Authentication System (`auth.cpp`)
* **Concepts Demonstrated:** Object-Oriented Programming (OOPs), Encapsulation, Data Security.
* **Key Feature:** Implemented low-level input stream manipulation using `<conio.h>` buffer readers (`_getch`) to handle real-time password masking (`*` display) without terminal echoing. Includes secure file parsing for account verification and sign-up uniqueness constraints.

### 2. Library Management System (`main.cpp`)
* **Concepts Demonstrated:** Advanced File Handling (I/O Streams), Structural Architecture, Database Simulation via Flat Files.
* **Key Feature:** Features a dynamic records deletion mechanism using dynamic text stream cloning (`ifstream`/`ofstream`), line-by-line validation, and physical file system swapping (`remove`/`rename`) with safe fallback pointers.

## 💻 Tech Stack & Fundamentals
* **Language:** C++ (Standard ISO C++)
* **Core paradigms:** Object-Oriented Programming (Classes, Objects, Access Specifiers), Advanced File I/O Streams, Memory Buffer Management.
